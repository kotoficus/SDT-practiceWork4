# Общее описание решения
Проект состоит из файлов, написанных на языке программирования Python и содержащих функции для вычисления площадей и периметров некоторых фигур (круга, квадрата).
# Обзор функций
## Файл circle.py
### Функция *area(r)*
```
def area(r):
    return math.pi * r * r
```
Принимает на вход радиус круга **r**, вычисляет (по форуле `S = πr^2`) и возвращает его площадь **S**.
- **Пример:**
|Входные данные|Выходные данные|
|--------|--------|
|0|0.0|
|1|3.141592653589793|
|2|12.566370614359172|
### Функция *perimeter(r)*
```
def perimeter(r):
    return 2 * math.pi * r
```
Принимает на вход радиус круга **r**, вычисляет (по форуле `P = 2πr`) и возвращает его периметр **P**.
- **Пример:**
|Входные данные|Выходные данные|
|--------|--------|
|0|0.0|
|1|6.283185307179586|
|2|12.566370614359172|

# Описание Unit тестов
Файл "test_circle.py" содержит класс, описывающий 6 групп тестов, нацеленных на проверку корректности работы функций area(r) и perimeter(r).
## Обзор функций
1. Функции test_area_normal_values и test_perimeter_normal_values проверяют корректность работы функций area(r) и perimeter(r) соответственно, передавая в качестве аргументов небольшие положительные числа.
2. Функции test_area_zero_value и test_perimeter_zero_value проверяют корректность работы функций area(r) и perimeter(r) соответственно, передавая в качестве аргумента 0.
3. Функции test_area_big_values и test_perimeter_big_values проверяют корректность работы функций area(r) и perimeter(r) соответственно, передавая в качестве аргументов положительные числа, большие, чем аргументы функций пункта 1.
